## Available Scripts (Use node 6.x or 7.x)

In the project directory, you can run:

### `npm run dev`

Runs the app in the development mode.<br>

### `npm run test`

Runs all tests.

### `npm run build`

Builds the app for production to the `dist` folder.<br>

### `npm run start`

To start the server in production.
